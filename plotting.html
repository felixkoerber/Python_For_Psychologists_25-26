Today we will have a look at data visualisation with matplotlib - the backbone of plotting in python. Check out seaborn a “wrapper” over matplotlib with a few “quality of life” convenience functions and a ton of examples.

%matplotlib inline

import numpy as np
import matplotlib.pyplot as plt  # pyplot is the base plotting module in python

Lets generate some data

a = np.random.random(100)  #
b = np.random.randn(100)  #

Matplotlib has a lot of low-level straight forward plotting functions, e.g., histograms

plt.hist(b)
plt.hist(a)

We can also plot lines very easily:

plt.plot(b)
plt.plot(a)

plt.plot(b,a)

this doesn’t really look like it makes much sense… So how about we first sort the values?

plt.plot(sorted(a), sorted(b))  # a gegen b in sortierter Reihenfolge

[<matplotlib.lines.Line2D at 0x12b4136d0>]

You can also draw lines by providing specific coordinates:

plt.plot((0, 1), (2, 3)) #(x1,x2),(y1,y2)

[<matplotlib.lines.Line2D at 0x12b39d3d0>]

You can add different styles to plt functions like hist, plt and scatter using arguments. For example we can add styles to the axes:

plt.plot(b, 'r+')  # red cross
plt.plot(a, color="blue", marker="o", linewidth=0)  # blue circle

plt.ylabel("nothing")
plt.xlabel("hello")

Text(0.5, 0, 'hello')

Usually, when we want to create more elaborate plots we first create the empty axes for our plot and subsequently fill it with the data of interest. In that case you first create the axes like so:

ax = plt.axes()
ax

<Axes: >

Which returns an empty set of axes.

then you can call the different plotting functions and styles directly to the axes.

ax = plt.axes()  # leere Axe

ax.plot(a - 5)  # line plot
ax.hist((b * 20) + 40)  # histogram (skaliert)
ax.scatter(20 + a * 20, b - 10, c="r")  # scatterplot

ax.set_ylabel("nothing")
ax.set_xlabel("all")

Text(0.5, 0, 'all')

You can add a legend by defining labels. In the end you can call plt.legend() to draw the legend.

ax = plt.axes()
ax.scatter(a, b, color='r', label="red dots")
ax.scatter(b, a, color='b', label="blue dots")
plt.legend()

<matplotlib.legend.Legend at 0x12b72d890>

Seaborn is a library for plotting that builds on top of matplotlib. It operates over dataframes and arrays that contain whole datasets. Let’s look at an example:

# Import seaborn
import seaborn as sns

# Apply the default theme
sns.set_theme()

# Load an example dataset
tips = sns.load_dataset("tips")
